## Organisation de la documentation

L'organisation et les standards de la documentation sont définis dans le document [Organisation de la Documentation](./Organisation%20de%20la%20Documentation%20-%20Course%20Facile.md), qui sert de référence centrale pour comprendre la structure et les conventions de la documentation.

## Contribution à la documentation

Pour contribuer à la documentation :
1. Suivez les conventions de nommage et de structure définies dans le document d'organisation
2. Mettez à jour la date de dernière modification en haut de chaque document
3. Assurez-vous que les liens entre les documents sont corrects
4. Vérifiez que votre contribution s'intègre bien dans la structure existante
